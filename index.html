Bootstrap RST

Bootstrap RST provides easy acces to bootstrap using the restructured text markup language.

Bootstrap RST offers an easy access to the bootstrap framework using the restructured text markup language. Bootstrap RST provides a set of new directives and roles (row, column, button, list, etc.) that allow seamless integration of the bootstrap framework. Have a look at the source of this page for example and see below for what is available so far.
